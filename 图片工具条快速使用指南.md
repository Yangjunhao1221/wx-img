# 图片工具条快速使用指南

## 🎯 功能简介

在布局拼图页面，点击画布上的图片即可弹出操作工具条，快速编辑图片。

## 📸 工具条界面

```
┌─────────────────────────────────────────────────┐
│  🖼️    🔄    ➕    ➖    🗑️                      │
│ 换一张  旋转  放大  缩小  删除                    │
└─────────────────────────────────────────────────┘
```

## 🔧 功能说明

### 1️⃣ 换一张 🖼️
- **功能**: 替换当前图片
- **操作**: 点击后选择新图片
- **效果**: 原位置图片被替换

### 2️⃣ 旋转 🔄
- **功能**: 旋转图片90度
- **操作**: 每次点击旋转90度
- **效果**: 0° → 90° → 180° → 270° → 0°

### 3️⃣ 放大 ➕
- **功能**: 放大图片
- **操作**: 每次点击放大10%
- **范围**: 100% - 200%

### 4️⃣ 缩小 ➖
- **功能**: 缩小图片
- **操作**: 每次点击缩小10%
- **范围**: 50% - 100%

### 5️⃣ 删除 🗑️
- **功能**: 删除图片
- **操作**: 点击后确认删除
- **效果**: 槽位变为空，可重新添加

## 💡 使用技巧

### 快速调整图片
1. 点击图片 → 工具条弹出
2. 选择需要的功能
3. 点击画布其他位置关闭工具条

### 组合使用
- 先旋转，再缩放
- 先缩放，再旋转
- 效果可叠加

### 导出保留
- 所有调整都会保存
- 导出图片包含所有效果
- 预览即所得

## ⚡ 快捷操作

| 操作 | 快捷方式 |
|------|---------|
| 显示工具条 | 点击图片 |

| 旋转90° | 工具条 → 旋转 |
| 放大10% | 工具条 → 放大 |
| 缩小10% | 工具条 → 缩小 |

## 🎨 最佳实践

### 图片旋转
- 横图变竖图：旋转90°或270°
- 倒置图片：旋转180°
- 恢复原状：旋转至0°

### 图片缩放
- 突出重点：放大主图片
- 平衡布局：缩小次要图片
- 适配布局：调整至合适大小

### 图片替换
- 不满意的图片：直接换一张
- 保持布局：只替换图片内容
- 快速调整：无需重新布局

## ⚠️ 注意事项

1. **点击空槽位**
   - 不会显示工具条
   - 直接添加图片

2. **缩放限制**
   - 最大200%（防止模糊）
   - 最小50%（防止太小）

3. **旋转角度**
   - 仅支持90度倍数
   - 适合大多数场景

4. **删除确认**
   - 需要确认操作
   - 防止误删

## 🌟 使用场景

### 场景1: 调整图片方向

```
问题：手机拍的照片方向不对

```

### 场景2: 突出主图
```
问题：想让某张图片更显眼
解决：点击图片 → 放大 → 调整至合适大小
```

### 场景3: 替换图片
```
问题：选错了图片
解决：点击图片 → 换一张 → 选择正确的图片
```

### 场景4: 删除多余图片
```
问题：不需要某张图片了
解决：点击图片 → 删除 → 确认
```
