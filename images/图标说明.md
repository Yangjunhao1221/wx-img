# 图标文件说明

## 需要的图标文件

为了完整支持tabBar图标，需要以下4个图标文件：

1. `collage.png` - 布局拼图默认图标 (32x32px)
2. `collage-active.png` - 布局拼图激活图标 (32x32px) 
3. `long.png` - 长图拼接默认图标 (32x32px)
4. `long-active.png` - 长图拼接激活图标 (32x32px)

## 图标要求

- 尺寸：32x32像素 或 64x64像素
- 格式：PNG格式，支持透明背景
- 颜色：默认图标建议使用灰色(#666)，激活图标使用蓝色(#007AFF)

## 临时解决方案

当前已移除app.json中的图标配置，小程序可以正常运行。
如需添加图标，请：

1. 准备符合要求的图标文件
2. 将图标文件放入images目录
3. 在app.json中恢复iconPath和selectedIconPath配置

## 图标设计建议

- 布局拼图图标：可以使用网格、拼图块等元素
- 长图拼接图标：可以使用长条、箭头等元素
- 保持图标风格统一，线条简洁清晰
